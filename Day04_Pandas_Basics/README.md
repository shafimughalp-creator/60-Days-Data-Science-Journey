## 🧠 Concepts Covered

| Concept | Description |
|---|---|
| **Series** | 1D labeled array — a single column of data |
| **DataFrame** | 2D table with rows and columns — like Excel |
| **df.head()** | View first 5 rows |
| **df.info()** | Column names, data types, null counts |
| **df.describe()** | Mean, min, max, std for numeric columns |
| **df["col"]** | Select a single column |
| **df.loc[]** | Select by label — inclusive |
| **df.iloc[]** | Select by position — exclusive |
| **pd.read_csv()** | Load data from a CSV file |
| **df.drop()** | Remove a column or row |
